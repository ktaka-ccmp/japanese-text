# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This repository contains Japanese language learning materials organized into graded lessons for different proficiency levels. The content focuses on practical conversational Japanese with phrase lists, vocabulary, and polite/casual dialogue examples.

## File Organization

Files follow a consistent naming pattern:
- `jpn_lv2_B##-B##_improved.txt` - Level 2 (Beginner+)
- `jpn_lv3_C##-C##_improved.txt` - Level 3 (Intermediate)
- `jpn_lv4_D##-D##_改善版.txt` - Level 4 (Upper Intermediate)
- `jpn_lv5_E##-E##_improved.txt` - Level 5 (Advanced)

Each filename contains lesson numbers (e.g., B01-B10 covers lessons B01 through B10).

## Content Structure

Each lesson file follows this standardized format:

1. **Lesson Header**: Topic identifier and theme in brackets
   ```
   ====================================
   B01: よろしくお願いします。
   【自己紹介をし、仕事について話す】
   ====================================
   ```

2. **フレーズリスト (Phrase List)**: Categorized phrases with bullet points and subcategories marked with ＊

3. **単語リスト (Vocabulary List)**: Relevant vocabulary for the lesson

4. **====== 丁寧版 ====== (Polite Version)**: Formal dialogue examples demonstrating the phrases in context

5. **Additional sections may include**:
   - ====== カジュアル版 ====== (Casual Version): Informal dialogue
   - 会話例 (Conversation Examples)
   - Grammar notes or cultural context

## Editing Guidelines

When modifying lesson files:

- **Maintain structural consistency**: All lessons should follow the same format (header → phrase list → vocabulary → dialogues)
- **Preserve Japanese formatting**: Keep proper use of punctuation marks (。、・「」【】), full-width characters, and spacing conventions
- **Respect the leveling system**: Content difficulty should match the level designation (lv2 = basic greetings, lv5 = complex business situations)
- **Keep phrase categorization**: Use ＊ for subcategory markers within phrase lists
- **Use consistent section headers**: Match the style of existing dividers (======, ------, etc.)
- **Maintain bilingual consistency**: Japanese text should be accompanied by appropriate context clues

## Level Distinctions

- **Level 2 (B-series)**: Basic introductions, daily routines, simple preferences
- **Level 3 (C-series)**: Extended conversations, expressing opinions, past/future tenses
- **Level 4 (D-series)**: Hypothetical situations, reasons and justifications, complex sentence structures
- **Level 5 (E-series)**: Formal business situations, nuanced expressions, honorific language (keigo)

## Character Encoding

All files use UTF-8 encoding to properly display Japanese characters. Ensure UTF-8 is preserved when editing.
