C31: 週末はいつも何をしますか。
【休日の過ごし方について話す】
====================================

フレーズリスト
--------------
・週末はいつも何をしますか。
・週末はたいてい家族でごはんを食べに行きます。
・週末はたいてい掃除をします。
・休みの日はいつも何をしますか。
・休みの日はたいてい買い物に行きます。
・休みの日はたいてい友達とテニスをします。
・先週末は何してた。
・先週はとてもおいしいレストランへ行ったよ。
・先週は家で、ゆっくりしてたよ。

単語リスト
----------
運動をする、ガーデニングをする、バーベキューをする、ゆっくりする
買い物に行く、ドライブに行く、ジムに行く、友達と遊びに行く
映画を見に行く、ごはんを食べに行く、テレビを見る


====== 丁寧版 ======

【休日の過ごし方について聞く】
山田：田中さんは週末、いつも何をされていますか。
田中：週末はたいてい家族でごはんを食べに行きます。
山田：いいですね。どこか決まったお店があるんですか。
田中：いえ、毎回違うところに行っています。

【休日の過ごし方について答える】
田中：山田さんは休みの日、何をされていますか。
山田：私はたいてい買い物に行きますね。
田中：ショッピングがお好きなんですね。
山田：はい、気分転換になりますから。

【最近の過ごし方について話す】
山田：先週末は何をされましたか。
田中：先週はとてもおいしいレストランへ行きました。
山田：それはよかったですね。どちらのお店ですか。
田中：新宿の「和食処 桜」という店です。おすすめですよ。


====== カジュアル版：友達との会話 ======

【休日の過ごし方について聞く】
山田：田中さん、週末いつも何してる？
田中：週末はたいてい家族でごはん食べに行くよ。
山田：いいね。どこか決まった店あるの？
田中：いや、毎回違うところ行ってる。

【休日の過ごし方について答える】
田中：山田さんは休みの日、何してる？
山田：俺はたいてい買い物行くかな。
田中：ショッピング好きなんだね。
山田：うん、気分転換になるしね。

【最近の過ごし方について話す】
山田：先週末は何してた？
田中：先週はめっちゃおいしいレストラン行ったよ。
山田：いいね。どこの店？
田中：新宿の「和食処 桜」ってとこ。おすすめだよ。

C32: ご注文はおきまりですか。
【飲食店で注文する】
====================================

フレーズリスト
--------------
・ご注文はおきまりですか。
・シーフードピザをください。
・ローストチキンをお願いします。
・申し訳ございません。本日、ローストチキンは終了しました。
・じゃあステーキをください。
・本日のパスタは何ですか。
・きのこの和風パスタでございます。
・フレッシュトマトのパスタと、きのこのクリームパスタからお選びください。
・お飲み物はいかがいたしましょうか。
・けっこうです。お水をもらえますか。
・デザートは何がありますか。
・チーズケーキとガトーショコラがございます。
・コーヒーと紅茶、どちらがいいですか。
・コーヒーはいつお持ちしましょうか。
・食事といっしょにお願いします。
・食後で。
・先に持ってきていただけますか。

単語リスト
----------
和風ハンバーグ、ローストビーフ、ステーキ
ジェラート、プリン、チーズケーキ


====== 丁寧版 ======

【注文を聞く・答える】
店員：ご注文はおきまりでしょうか。
客：はい、シーフードピザをお願いします。
店員：かしこまりました。サイズはいかがいたしますか。
客：Lサイズでお願いします。

【品切れの対応】
店員：申し訳ございません。本日、ローストチキンは終了いたしました。
客：そうですか。では、ステーキをお願いします。
店員：かしこまりました。焼き加減はいかがいたしますか。
客：ミディアムでお願いします。

【飲み物とデザートを注文する】
店員：お飲み物はいかがいたしましょうか。
客：お水をいただけますか。
店員：かしこまりました。デザートはいかがでしょうか。
客：チーズケーキをお願いします。
店員：コーヒーはいつお持ちいたしましょうか。
客:食後でお願いします。


====== カジュアル版：友達との会話 ======

【注文を聞く・答える】
店員：ご注文おきまりですか。
客：はい、シーフードピザください。
店員：サイズはどうしますか。
客：Lサイズで。

【品切れの対応】
店員：すみません。今日、ローストチキン終了しちゃいました。
客：そっか。じゃあ、ステーキで。
店員：わかりました。焼き加減どうします？
客：ミディアムで。

【飲み物とデザートを注文する】
店員：飲み物どうしますか。
客：お水もらえる？
店員：わかりました。デザートは？
客：チーズケーキください。
店員：コーヒーいつ持ってきます？
客：食後でいいです。

C33: 買い物に行くには、どこがおすすめ。
【行きたい場所、おすすめしたい場所について話す】
====================================

フレーズリスト
--------------
・観光したいんだけど、どこかおすすめのところはある。
・浅草がおすすめだよ。
・東京タワーにはもう行った。
・ううん、まだだよ。
・夜がおすすめだよ。
・じゃあ、行ってみるよ。
・ごはんを食べに行くのに、どこかいいところはある。
・横浜の中華街がおすすめだよ。
・「N's ダイニング」がおすすめだよ。
・買い物するには、どこへ行けばいい。
・原宿がいいよ。

単語リスト
----------
東京ドーム、中華街、観光する、買い物に行く／買い物する
夜景を見る、富士山を見る、飲みに行く


====== 丁寧版 ======

【観光スポットについて聞く】
李：すみません、観光したいのですが、どこかおすすめの場所はありますか。
田中：そうですね、浅草がおすすめですよ。
李：浅草ですか。どんなところですか。
田中：伝統的な雰囲気があって、お寺や商店街があります。

【行ったことがあるか確認する】
田中：東京タワーにはもう行かれましたか。
李：いいえ、まだです。
田中：それでしたら、ぜひ夜に行ってみてください。夜景がとてもきれいですよ。
李：そうですか。じゃあ、行ってみます。

【食事の場所について聞く】
李：おいしいごはんを食べに行くのに、どこかいい場所はありますか。
田中：横浜の中華街がおすすめです。
李：中華街ですか。たくさんお店がありそうですね。
田中：はい、色々な種類の料理が楽しめますよ。


====== カジュアル版：友達との会話 ======

【観光スポットについて聞く】
李：ねえ、観光したいんだけど、どこかおすすめある？
田中：浅草がいいよ。
李：浅草？どんなとこ？
田中：伝統的な雰囲気あって、お寺とか商店街があるよ。

【行ったことがあるか確認する】
田中：東京タワーにはもう行った？
李：ううん、まだ。
田中：じゃあ、夜行ってみて。夜景めっちゃきれいだから。
李：そっか。じゃあ、行ってみる。

【食事の場所について聞く】
李：おいしいごはん食べに行くのに、どこかいいとこある？
田中：横浜の中華街がおすすめだよ。
李：中華街？たくさん店ありそうだね。
田中：うん、色々な料理が楽しめるよ。

C34: A席を2枚お願いします。
【イベントのチケットを買う】
====================================

フレーズリスト
--------------
・すみません。野外コンサートのチケットがほしいのですが。
・ありがとうございます。ご希望のお日にちは。
・8月20日がいいんですけど。
・お席はどうされますか。
・A席を2枚お願いします。
・申し訳ございません。A席は売切れでございます。
・S席はどうですか。
・S席で空きはありますか。
・S席でしたら空いております。
・S席はいくらですか。
・9,000円です。
・じゃ、それでお願いします。
・そうですか。ちょっと考えます。

単語リスト
----------
コートサイド、センター席、アリーナ席、スタンド席
ホーム側、アウェイ側、自由席、立見席


====== 丁寧版 ======

【チケットを申し込む】
客：すみません。野外コンサートのチケットがほしいのですが。
店員：ありがとうございます。ご希望のお日にちはいつでしょうか。
客：8月20日でお願いします。
店員:かしこまりました。お席はどうされますか。

【席を選ぶ】
客：A席を2枚お願いします。
店員：申し訳ございません。A席は売切れでございます。
客：S席はどうですか。空きはありますか。
店員：S席でしたら空いております。

【料金を確認する】
客：S席はいくらですか。
店員：お一人様9,000円でございます。2枚で18,000円になります。
客：わかりました。それでお願いします。
店員：かしこまりました。お支払いは現金とカードどちらになさいますか。


====== カジュアル版：友達との会話 ======

【チケットを申し込む】
客：すみません。野外コンサートのチケットほしいんですけど。
店員：ありがとうございます。希望の日にちは？
客：8月20日で。
店員：わかりました。席はどうします？

【席を選ぶ】
客：A席2枚ください。
店員：すみません。A席売り切れちゃって。
客：S席は？空いてる？
店員：S席なら空いてますよ。

【料金を確認する】
客：S席いくら？
店員：一人9,000円です。2枚で18,000円ですね。
客：わかった。それで。
店員：わかりました。支払い、現金とカードどっちがいいですか。

C35: 生まれも育ちも大阪です。
【生まれ育った場所について話す】
====================================

フレーズリスト
--------------
・出身はどこですか。
・どこで育ちましたか。
・どこに住んでいますか。
・東京です。
・ずっと東京ですか。
・出身は博多です。
・どんなところですか。
・大きな町で、ラーメンが有名です。
・いなかで、田んぼが多いです。
・人が多くて、にぎやかなところだよ。
・静かなところだよ。
・大きなショッピングモールがあって、便利なところだよ。

単語リスト
----------
にぎやかな、便利な、静かな、有名な
人が少ない、公園が多い、コンビニがない
都会、郊外、田舎


====== 丁寧版 ======

【場所の特徴を説明する】
山田：大阪はどんなところですか。
田中：大きな都会で、とてもにぎやかなところです。
山田：食べ物もおいしいですよね。
田中：はい、たこ焼きやお好み焼きが有名です。

【現在住んでいる場所について話す】

山田：今は横浜に住んでいます。
田中：横浜はどんなところですか。

【現在住んでいる場所について話す】

田中：横浜ってどんなとこ？
山田：静かで、大きなショッピングモールとかあって便利なとこだよ。

C36: ご出発予定は何日ですか。
【旅行の予約をする】
====================================

フレーズリスト
--------------
・エクスプレストラベルでございます。
・東京3日間のプランを予約したいんですが。
・ご出発予定は何日ですか。
・1月22日です。
・何名様ですか。
・3人です。
・申し訳ございません。22日ご出発の分は空きがございません。
・そうですか。じゃ、23日ご出発の分はまだ空きがありますか。
・そうですか、ちょっと考えます。
・そうですか、友達と相談します。
・またあらためます。
・23日ご出発の分でしたら、空きがございます。
・じゃ、23日出発の分でお願いします。
・同じくらいの予算で別のものはありますか。

単語リスト
----------
日帰り、3日間、5日間、10日間


====== 丁寧版 ======

【旅行を予約する】
店員：エクスプレストラベルでございます。
客：東京3日間のプランを予約したいのですが。
店員：ありがとうございます。ご出発予定は何日でしょうか。
客：1月22日です。

【人数と空き状況を確認する】
店員：何名様でいらっしゃいますか。
客：3人です。
店員：少々お待ちください。申し訳ございません。22日ご出発の分は空きがございません。
客：そうですか。23日ご出発の分はまだ空きがありますか。

【予約を決定する】
店員：23日ご出発の分でしたら、空きがございます。
客：わかりました。では、23日出発の分でお願いします。
店員：かしこまりました。それでは、お名前とご連絡先をお願いいたします。


====== カジュアル版：友達との会話 ======

【旅行を予約する】
店員：エクスプレストラベルです。
客：東京3日間のプラン予約したいんですけど。
店員：ありがとうございます。出発予定いつですか。
客：1月22日です。

【人数と空き状況を確認する】
店員：何人ですか。
客：3人です。
店員：ちょっと待ってください。すみません、22日出発の分、空きないんです。
客：そうですか。23日出発の分はまだある？

【予約を決定する】
店員：23日出発の分なら空いてます。
客：わかりました。じゃ、23日出発の分で。
店員：わかりました。じゃ、名前と連絡先教えてください。

C37: お手洗いはどちらですか。
【建物の中で場所をたずねる】
====================================

フレーズリスト
--------------
・すみません。お手洗いはどちらですか。
・7階にございます。
・エレベーターを出てまっすぐ行った左手にございます。

・まっすぐ行ってつきあたりを左です。
・山田さんのお部屋はどちらですか。
・一番おくの部屋です。
・エレベーターで6階に行ってください。
・何階ですか。／何階にありますか。

単語リスト

エスカレーター、会議室、カフェ／喫茶店
通路、つきあたり、おく、正面

C38: 田村さんも来ませんか。
【人をさそう】
====================================

フレーズリスト
--------------
・今度、クリスマスパーティーをするんですが、来ませんか。
・いいですね、行きます。
・行きたいです。
・すみません、その日はちょっと...
・またさそってください。
・花見をするんですが、メアリーさんも来ませんか。
・いいですね。どこでするんですか。
・大阪城公園です。
・何時からですか。
・1時からの予定です。
・誰が来るんですか。
・木下君と野村さんとジョンさんです。

単語リスト
----------
（お）誕生日会、（お）花見、送別会、歓迎会
新年会、忘年会、クリスマスパーティー


====== 丁寧版 ======

【イベントにさそう】
田中：今度、クリスマスパーティーをするのですが、来ませんか。

【詳細を伝える】
田中：場所は渋谷の「パーティールーム スター」です。
山田：何時からですか。
田中：7時からの予定です。
山田：他に誰が来るんですか。
田中：木下さんと野村さんとジョンさんです。

【断る】
田中：今度、忘年会をするのですが、来ませんか。
山田：すみません、その日はちょっと予定がありまして...
田中：そうですか。残念です。
山田：申し訳ございません。またさそってください。

補足：各エピソードの活用方法
====================================

【丁寧版とカジュアル版の使い分け】
・丁寧版：ビジネスシーン、サービス業での会話、初対面の人、公共の場
・カジュアル版：友達同士、同僚との気軽な会話、家族との会話

【中国人学習者向けの特徴】
・漢字を積極的に使用（中国人学習者は漢字の認識が速い）
・実用的な場面を重視（休日、食事、旅行、買い物など）
・丁寧語とカジュアル語の違いを明確に提示

【学習のポイント】
1. まず基本フレーズを覚える
2. 丁寧版で正式な表現を学ぶ
3. カジュアル版で日常会話を練習する
4. 状況に応じて使い分けられるようにする
5. 実際の場面を想定して練習する

【練習方法】
1. ロールプレイング：二人一組で役を交代しながら練習
2. 状況設定：休日、レストラン、観光、ショッピングなど具体的な場面を想定
3. 単語の入れ替え：基本文型を使って単語を変えて練習
4. 自由会話：学んだ表現を使って自由に会話を展開する
5. 録音練習：自分の発音を録音して確認する
