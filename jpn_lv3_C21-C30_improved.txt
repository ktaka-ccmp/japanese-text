C21: テレビが映らないんです。
【修理を依頼する】
====================================

フレーズリスト
--------------
・今日、修理に来てもらえますか。
・水道管から水が漏れているので、修理をお願いしたいんですが。
・かしこまりました。
・15時はいかがですか。
・テレビが映らないんです。
・見に来てもらえますか。
・すみません。今日は予約がいっぱいなんです。
・明日の朝はどうですか。
・今日中にお願いできませんか。
・今日の17時でしたら。
・いくらぐらいかかりますか。
・そうですね。10,000円から20,000円ぐらいです。
・わかりました。お願いします。
・ちょっと高いですね。
・また改めます。

単語リスト
----------
水道管から水が漏れている、テレビが映らない
トイレが詰まっている、玄関の鍵が壊れている
冷蔵庫から変な音がする、修理、冷蔵庫


====== 丁寧版 ======

【修理を依頼する】
王さん：もしもし、エアコンの修理をお願いしたいのですが。
業者：かしこまりました。どのような状態でしょうか。
王さん：エアコンが全く効かなくなってしまいまして。
業者：それは大変ですね。いつ頃お伺いできますか。

【日時を決める】
業者：今日の15時はいかがでしょうか。
王さん：申し訳ございません、その時間は留守にしております。
業者：それでは、明日の午前中はいかがですか。
王さん：明日の10時でお願いできますでしょうか。

【料金について】
王さん：修理費用はどのくらいかかりますか。
業者：状況によりますが、10,000円から20,000円程度です。
王さん：わかりました。それではお願いします。
業者：かしこまりました。明日10時にお伺いいたします。


====== カジュアル版：友達との会話 ======

【修理について話す】
王さん：エアコン壊れちゃってさ。
友達：マジで？どうしたの？
王さん：全然効かなくなった。
友達：それはやばいね。修理頼んだ？

【業者とのやり取り】
王さん：今日修理来てもらえるか聞いたんだけど。
友達：で、どうだった？
王さん：今日はいっぱいだって。明日来てくれるらしい。
友達：そっか。料金いくらくらい？

【料金について】
王さん：1万から2万くらいだって。
友達：まあ、そんなもんか。
王さん：高いけど、仕方ないよね。
友達：うん、夏だし、早く直した方がいいよ。

C22:「クレイジー・タウン」はどう？
【見る映画を選ぶ】
====================================

フレーズリスト
--------------
・どの映画を見たい？
・何を見る？
・「ビースト」を見ない？
・ごめん、ホラーは好きじゃないんだ。
・ごめん、ホラーはあまり見ないんだ。
・それ、あまり面白くないらしいよ。
・「クレイジー・タウン」はどう？
・どんな映画？
・コメディだよ。
・アメリカの家族もののコメディだよ。
・面白いらしいよ。
・面白そうだね。
・じゃあ、クレイジー・タウンにしよう。
・いいね、それにしよう。

単語リスト
----------
コメディ、アクション、SF、ホラー、スリラー
恋愛もの/ラブストーリー、ドラマ


====== 丁寧版 ======

【映画を選ぶ】
田中：王さん、今週末映画を見に行きませんか。
王さん：いいですね。どんな映画がいいですか。
田中：アクション映画はいかがですか。
王さん：すみません、アクションはあまり得意ではないんです。

【別の提案をする】
田中：それでは、コメディはどうですか。「クレイジー・タウン」という映画が評判がいいそうです。
王さん：どのような内容ですか。
田中：アメリカの家族の日常を描いたコメディです。
王さん：面白そうですね。それにしましょう。

【決定する】
田中：それでは、土曜日の午後2時からの回はいかがですか。
王さん：はい、それで大丈夫です。
田中：では、チケットを予約しておきます。
王さん：ありがとうございます。楽しみにしています。


====== カジュアル版：友達との会話 ======

【映画を選ぶ】
友達：今週末映画見に行かない？
王さん：いいね。何見る？
友達：アクションとかどう？
王さん：ごめん、アクションあんまり好きじゃないんだ。

【別の提案をする】
友達：じゃあ、コメディは？「クレイジー・タウン」って映画、評判いいらしいよ。
王さん：どんな映画？
友達：アメリカの家族の話。コメディだよ。
王さん：面白そうだね。それにしよう。

【決定する】
友達：土曜日の2時からの回どう？
王さん：OK。それで。
友達：じゃあ、チケット予約しとくね。
王さん：ありがとう。楽しみ。

C23: 週末はどうだった？
【最近の出来事について話す】
====================================

フレーズリスト
--------------
・週末はどうだった？
・家にいたよ。
・土曜は野球を見に行ったよ。
・日曜はドライブに行ったよ。
・映画はどうだった？
・まあまあだった。
・休みの日は何をしたの？
・友達とご飯を食べに行ったよ。
・お店はどうだった？
・料理も美味しかったし、雰囲気も良かったよ。
・美味しかったけど、少し高かったよ。
・沖縄は楽しかった？
・楽しかったよ。
・水族館に行ったよ。
・どうだった？
・とても綺麗で良かったよ。

単語リスト
----------
最高/最悪、美味しい/不味い、高い/安い
面白い/つまらない、いい/悪い


====== 丁寧版 ======

【週末の出来事を聞く】
田中：王さん、週末はいかがでしたか。
王さん：友人と京都に行ってきました。
田中：それは良いですね。京都はどうでしたか。
王さん：とても良かったです。紅葉が綺麗でした。

【詳しく聞く】
田中：どちらを訪れましたか。
王さん：清水寺と金閣寺に行きました。
田中：食事はされましたか。
王さん：はい、京料理を食べました。とても美味しかったです。

【感想を述べる】
田中：それは素晴らしいですね。また行きたいですか。
王さん：はい、ぜひまた行きたいです。次は桜の季節に行きたいと思っています。
田中：それは良い計画ですね。
王さん：田中さんも一緒にいかがですか。


====== カジュアル版：友達との会話 ======

【週末の出来事を聞く】
友達：週末どうだった？
王さん：京都行ってきた。
友達：いいね。どうだった？
王さん：めっちゃ良かった。紅葉が綺麗だった。

【詳しく聞く】
友達：どこ行ったの？
王さん：清水寺と金閣寺。
友達：ご飯は？
王さん：京料理食べた。めちゃくちゃ美味しかった。

【感想を述べる】
友達：いいなあ。また行きたい？
王さん：うん、また行きたい。次は桜の時期に。
友達：それいいね。
王さん：一緒に行かない？

C24: 伝言を承りましょうか。
【電話をかけ、伝言を残す】
====================================

フレーズリスト
--------------
・お電話ありがとうございます。ABC旅行社でございます。
・私、ジョンソンと申しますが、山本さんは、いらっしゃいますか。
・山本は席を外しております。
・よろしければ、ご伝言をお預かりいたしましょうか。
・別の電話に出ておりまして…
・あいにく、休憩に出ております。
・会議に出ております。
・結構です。改めてお電話します。
・会議の後、お電話くださいとお伝えください。
・山本は本日お休みをいただいております。
・失礼ですが、もう一度、お名前とご連絡先をいただけますでしょうか。
・ジョンソンと申します。
・060-426-3957です。
・お電話番号を確認させていただきます。
・060-426-3957、ジョンソン様ですね。
・はい、そうです。よろしくお願いします。
・失礼いたします。

単語リスト
----------
外出中、休憩中、会議中


====== 丁寧版 ======

【電話を受ける】
受付：お電話ありがとうございます。田中商事でございます。
王さん：私、王と申しますが、鈴木部長はいらっしゃいますでしょうか。
受付：申し訳ございません。鈴木は現在会議中でございます。
王さん：そうですか。いつ頃お戻りでしょうか。

【伝言を頼む】
受付：1時間後には会議が終わる予定です。よろしければご伝言を承りましょうか。
王さん：それではお願いします。会議が終わりましたら、折り返しお電話いただけますようお伝えください。
受付：かしこまりました。恐れ入りますが、お名前とご連絡先をお願いできますでしょうか。
王さん：王と申します。電話番号は090-1234-5678です。

【確認する】
受付：復唱させていただきます。090-1234-5678、王様ですね。
王さん：はい、そうです。よろしくお願いいたします。
受付：かしこまりました。鈴木に必ずお伝えいたします。
王さん：ありがとうございます。失礼いたします。


====== カジュアル版：知人の会社への電話 ======

【電話を受ける】
受付：お電話ありがとうございます。田中商事です。
王さん：王ですけど、鈴木さんいますか。
受付：すみません、今会議中なんです。
王さん：そうですか。いつ頃終わりますか。

【伝言を頼む】
受付：たぶん1時間後くらいです。伝言しておきましょうか。
王さん：じゃあ、会議終わったら電話くださいって伝えてもらえますか。
受付：わかりました。お名前と電話番号教えてもらえますか。
王さん：王です。090-1234-5678です。

【確認する】
受付：090-1234-5678、王さんですね。
王さん：はい、そうです。よろしくお願いします。
受付：わかりました。鈴木さんに伝えておきます。
王さん：ありがとうございます。

C25: 予約している山口です。
【ホテルに泊まる】
====================================

フレーズリスト
--------------
・いらっしゃいませ。
・ようこそお越しくださいました。
・予約している山口です。
・チェックインをお願いします。
・お待ちしておりました。
・ご夕食は何時になさいますか。
・7時半にお願いします。
・大浴場は何階ですか。
・1階でございます。
・チェックアウトは何時ですか。
・10時でございます。
・お風呂は何時までですか。
・露天風呂は、朝9時から夜11時まででございます。
・大浴場は24時間ご利用いただけます。
・お部屋の鍵でございます。
・ごゆっくりお過ごしくださいませ。

単語リスト
----------
客室、自動販売機/売店/ホテルショップ/フロント
大浴場/露天風呂、レストラン/バー、駐車場
美容室/理容室、旅館


====== 丁寧版 ======

【チェックインする】
フロント：いらっしゃいませ。ようこそお越しくださいました。
王さん：予約している王と申します。チェックインをお願いします。
フロント：お待ちしておりました。少々お待ちくださいませ。
王さん：はい。

【施設について聞く】
王さん：大浴場は何階にございますか。
フロント：1階でございます。24時間ご利用いただけます。
王さん：ありがとうございます。朝食は何時からですか。
フロント：朝7時から9時半までレストランでご用意しております。

【チェックアウトについて】
王さん：チェックアウトは何時ですか。
フロント：10時でございます。
王さん：わかりました。
フロント：こちらがお部屋の鍵でございます。5階の502号室です。ごゆっくりお過ごしくださいませ。


====== カジュアル版：友達との会話（ホテルで） ======

友達：はい。
王さん：部屋何階？

【施設について聞く】
王さん：大浴場どこ？
友達：1階だって。24時間入れるみたい。
王さん：いいね。朝ご飯は？
友達：7時から9時半までらしいよ。

【チェックアウトについて】
王さん：チェックアウト何時まで？
友達：10時だって。
王さん：わかった。
友達：じゃあ、部屋行こう。

C26: コートはいりますか？
【気候に合った服を勧める】
====================================

フレーズリスト
--------------
・7月の香港はどんな気候？
・夏だからとても暑いよ。
・どんな服を着て行けばいい？
・Tシャツで大丈夫だよ。
・冬に北海道に行く予定なんだけど、どんな服を持って行けばいいと思う？
・ダウンと手袋はあった方がいいよ。
・帽子はいる？
・うん、あった方がいいよ。
・ううん、いらないと思うよ。
・雨がよく降るから、傘を持って来た方がいいよ。

単語リスト
----------
マフラー、帽子、セーター、サングラス、水着
サンダル、ブーツ、傘、レインコート、手袋


====== 丁寧版 ======

【気候について聞く】
王さん：来月北海道に行く予定なのですが、どのような気候でしょうか。
田中：1月は非常に寒いです。気温はマイナス5度から10度くらいです。
王さん：それは寒いですね。どのような服装がいいでしょうか。
田中：厚手のダウンジャケットが必要です。

【持ち物について】
王さん：他に何か必要なものはありますか。
田中：手袋とマフラーは必須です。帽子もあった方がいいですよ。
王さん：ブーツは必要ですか。
田中：はい、雪道を歩くので、防水のブーツをお勧めします。

【アドバイスをする】
田中：それから、ホッカイロも持って行かれるといいですよ。
王さん：ホッカイロですか。
田中：はい、使い捨てのカイロです。とても便利ですよ。
王さん：わかりました。ありがとうございます。


====== カジュアル版：友達との会話 ======

【気候について聞く】
王さん：来月北海道行くんだけど、どんな気候？
友達：1月？めちゃくちゃ寒いよ。マイナス5度から10度くらい。
王さん：やばいね。どんな服着てけばいい？
友達：厚手のダウンは必須。

【持ち物について】
王さん：他に何かいる？
友達：手袋とマフラーは絶対。帽子もあった方がいいよ。
王さん：ブーツは？
友達：うん、雪道歩くから防水のやつがいいよ。

【アドバイスをする】
友達：あと、ホッカイロも持ってくといいよ。
王さん：ホッカイロ？
友達：使い捨てのカイロ。めっちゃ便利だよ。
王さん：わかった。ありがとう。

C28: スーパーの向かいにあります。
【街中の施設の位置について説明する】
====================================

フレーズリスト
--------------
・本屋はどこにありますか。
・本屋はコンビニの隣ですか。
・病院は銀杏通りにありますか。
・レストランとカフェの間にあります。
・一つ目の角にあります。
・コンサートホールに行きたいんですが。
・海岸通りにあります。
・映画館はどこにありますか。
・銀杏通りの郵便局の左隣にあります。
・郵便局と図書館の間にあります。
・近くに駐車場はありますか。
・はい、市役所の向かいにあります。
・いいえ、ありません。
・レストランの向かいの建物ですか。
・はい、そうです。
・いいえ、あれは郵便局です。

単語リスト
----------
右隣/左隣、角、向かい、斜め前


====== 丁寧版 ======

【場所を尋ねる】
王さん：すみません、市役所はどこにありますか。
通行人：この通りをまっすぐ行って、3つ目の角を右に曲がってください。
王さん：3つ目の角を右ですね。
通行人：はい、そうです。市役所は銀行の向かいにあります。

【詳しく聞く】
王さん：駐車場はありますか。
通行人：はい、市役所の隣に公共駐車場があります。
王さん：ありがとうございます。徒歩でどのくらいかかりますか。
通行人：ここから10分ほどです。

【確認する】
王さん：それでは、まっすぐ行って3つ目の角を右ですね。
通行人：はい、そうです。大きな建物なので、すぐわかると思います。
王さん：ありがとうございました。
通行人：どういたしまして。


====== カジュアル版：友達との会話 ======

補足：各エピソードの活用方法
====================================

【丁寧版とカジュアル版の使い分け】
・丁寧版：ビジネスシーン、サービス業での会話、初対面の人、公共の場
・カジュアル版：友達同士、同僚との気軽な会話、家族との会話

【中国人学習者向けの特徴】
・漢字を積極的に使用（中国人学習者は漢字の認識が速い）
・修理、映画、旅行、返品など日常で使う実用的な場面を重視
・丁寧語とカジュアル語の違いを明確に提示

【学習のポイント】
1. まず基本フレーズを覚える
2. 丁寧版で正式な表現を学ぶ
3. カジュアル版で日常会話を練習する
4. 状況に応じて使い分けられるようにする
5. 実際の場面を想定して練習する

【練習方法】
1. ロールプレイング：二人一組で役を交代しながら練習
2. 状況設定：修理依頼、映画選び、ホテル、返品など具体的な場面を想定
3. 単語の入れ替え：基本文型を使って単語を変えて練習
4. 自由会話：学んだ表現を使って自由に会話を展開する
5. 録音練習：自分の発音を録音して確認する
