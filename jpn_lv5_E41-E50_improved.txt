E41: 失礼だから人を指差さないほうがいいよ。
【習慣とマナーについて話す】
====================================

フレーズリスト
--------------
＊習慣について話す
・日本では、靴のまま家にあがってはいけないんでしょ。
・家にあがる時は靴をぬぐものなんだ。
・私の国にはそんな習慣はないよ。

＊食事のマナーについて
・うどんやそばを食べる時に音を立てても大丈夫なの。
・大丈夫だよ。むしろその方がいいとされているんだ。音を立てた方が美味しそうでしょ。
・私の国では、音を立てるのは行儀が悪いとされているよ。

＊マナーについて注意する
・道を歩きながら何かを食べるのは、行儀が悪いとされているんだ。
・私の国でもそうだよ。
・失礼だから、人を指差さない方がいいよ。
・それ、失礼だからやめた方がいいよ。

単語リスト
----------
歩きタバコ、パーティーに遅れて来る
行儀がいい、行儀が悪い、音を立てる、音を立てない


====== 丁寧版 ======

【日本の習慣について説明する】
田中：日本では、家に上がる時は靴を脱ぐんですよ。
鈴木：そうなんですか。私の国ではそのような習慣はありませんでした。
田中：文化の違いですね。日本では家の中を清潔に保つ習慣があるんです。
鈴木：なるほど。勉強になります。

【食事のマナーについて】
鈴木：麺類を食べる時、音を立てても大丈夫なんですか。
田中：はい、日本では音を立てて食べることが美味しさの表現とされています。
鈴木：興味深いですね。私の国では音を立てるのは行儀が悪いとされています。
田中：文化によって違いがあって面白いですね。

【マナーについて注意する】
田中：すみません、日本では人を指差すのは失礼とされています。
鈴木：申し訳ございません。知りませんでした。
田中：いえいえ、文化の違いですから。気をつけてくださいね。
鈴木：ありがとうございます。以後気をつけます。

E42: もっと勉強しておくべきだった。
【後悔していることについて話す】
====================================

フレーズリスト
--------------
＊後悔について話す
・あの時、あの服を買っておくべきだったな。欲しかったな。
・こんなにいっぱいなら、お店、予約しておけばよかったね。
・あーあ、ビデオとっておけばよかった。見たかったな。
・残念だね。

＊学生時代の後悔
・学生の時、もっと勉強するべきだった。
・勉強していたら、今こんなに困らなかったのに。
・本当に、そう思うよ。

＊人間関係の後悔
・あの時、ちゃんと謝っておくべきだった。
・謝っていたら、また友達に戻れたのに。
・今からでも間に合うよ。

＊失敗の後悔
・何で、結婚指輪をなくしてしまったんだろう。あんなに飲むんじゃなかった。
・あの最後の一杯をやめておけばよかった。
・本当に後悔しているよ。
・どうしてあんなことを言ってしまったんだろう。つい、かっとなって言ってしまったんだ。
・あんなこと、言うべきじゃなかった。悔やんでも悔やみきれないよ。
・もう過ぎたことだよ。
・もっと注意深く、運転していたらなあ。
・わかっていたはずなのに、どうして･･･
・ちゃんと／きちんと量をはかればよかった。そうすれば失敗しなかったのに。
・今さらどうすることもできないよ。
・次から気をつけるしかないよ。

単語リスト
----------
後悔する、〜すべきだった、〜しておけばよかった
〜していたら〜のに、残念、悔やむ


====== 丁寧版 ======

【過去の選択について後悔する】
田中：あの時、もっと勉強しておくべきでした。
鈴木：そうですね。でも今からでも遅くはないと思いますよ。
田中：本当にそう思います。あの時勉強していたら、今こんなに困らなかったのに。
鈴木：過去は変えられませんが、これからの選択は変えられますよ。

【買い物の後悔】
鈴木：あの服、買っておけばよかったですね。
田中：そうですね。でも、また似たようなものが見つかるかもしれません。
鈴木：そうだといいんですが。本当に欲しかったです。
田中：次回は即決するようにしましょう。

【人間関係の後悔】
田中：あの時、ちゃんと謝っておくべきでした。
鈴木：まだ遅くないと思いますよ。今からでも謝ってはいかがですか。
田中：そうですね。謝っていれば、また友達に戻れたかもしれません。
鈴木：ぜひ連絡を取ってみてください。


====== カジュアル版：友達との会話 ======

E43: 現代人の食生活は･･･
【現代社会の問題について話す】
====================================

フレーズリスト
--------------
＊人に意見を求める
・現代人はファーストフードを食べすぎだと思いませんか。
・ジャンクフードは体に悪いと思いますか。
・近頃の子供たちは、小さい頃からパソコンを使いすぎだと思いますか。
・暴力的なシーンが多い映画やドラマに関して、どのように思いますか。
・最近のスマートフォン人気についてどのようにお考えですか。

＊同意したり反対の意見を述べたりする
・間違いなく、そうですよね。／そのとおりです。
・おっしゃるとおりだと思います。
・確実に悪影響を与えていると思います。
・恐らく。
・私は気になりませんが･･･
・全く食べてはいけないというわけではなく、毎日食べ続けることが問題だと思います。
・全てのテレビ番組が子供に悪影響だとは思いません。
・そうかもしれませんが…全てが悪いとは限らないのではないでしょうか。
・食べる人の気持ちはわかります。体に悪いとわかっていてもつい食べてしまうものですから。
・悪いとわかっていてもなかなかやめられませんよね。
・パソコンが使えないと将来困るので、多少多くてもいいと思います。

単語リスト
----------
賛否両論、運動不足、良い／悪い影響を与える、害を与える


====== 丁寧版 ======

【現代社会の問題について意見を求める】
田中：現代人はファーストフードを食べすぎだと思いませんか。
鈴木：おっしゃるとおりだと思います。健康への影響が心配ですね。
田中：そうですね。毎日食べ続けることが特に問題だと思います。
鈴木：確かに。たまに食べる分には問題ないかもしれませんね。

【子供とテクノロジーについて】
鈴木：最近の子供たちは、小さい頃からパソコンを使いすぎだと思いますか。
田中：そうかもしれませんが、パソコンが使えないと将来困ると思います。
鈴木：それはそうですね。適度な使用が大切ということでしょうか。
田中：はい、バランスが重要だと思います。

【メディアの影響について】
田中：暴力的なシーンが多い映画やドラマについてどう思われますか。
鈴木：確実に悪影響を与えている部分はあると思います。
田中：全てが悪いとは限らないのではないでしょうか。
鈴木：そうかもしれませんね。内容や見方によりますね。

E44: テレビがない生活なんて考えられません。
【ないと困るものについて話す】
====================================

フレーズリスト
--------------
＊ないと困るものについて話す
・私は音楽がなかったら、生きていけません。
・音楽のおかげで心が豊かになります。
・もう携帯（電話）なしでは生きていけていけません。
・携帯電話がないと、待ち合わせに遅れそうな時に連絡することができなくて困ります。
・とても便利な反面、事故の原因になる恐れもあります。

＊インターネットについて
・インターネットのおかげで、とても便利に生活することができます。
・インターネットのいいところは、離れている友達や家族といつでもメールをしたりできるところです。
・その一方で、犯罪などに利用される恐れがあるのも事実です。

＊様々な意見
・テレビがないと困ります。楽しみがなくなります。
・私はテレビのない生活なんて、考えられません。
・私は田舎に住んでいるので、車がないと困ります。
・もし車がなかったら、とても不便だと思います。
・昔はなかったわけだし、自転車や電車があれば、車はなくても大丈夫じゃありませんか。

単語リスト
----------
環境汚染、事故の原因、発明品


====== 丁寧版 ======

【必需品について話す】
田中：鈴木さんにとって、なくてはならないものは何ですか。
鈴木：私にとっては携帯電話ですね。これがないと生活できません。
田中：わかります。連絡を取るのに本当に便利ですよね。
鈴木：はい、待ち合わせの時など特に助かります。

【音楽の重要性について】
鈴木：田中さんは音楽がお好きですよね。
田中：はい、音楽がなかったら生きていけないくらいです。
鈴木：それほどですか。音楽にはそんなに魅力があるんですね。
田中：ええ、音楽のおかげで心が豊かになります。

【車の必要性について議論する】
田中：私は田舎に住んでいるので、車がないと困ります。
鈴木：そうでしょうね。交通機関が少ないと大変ですよね。
田中：はい、もし車がなかったら、とても不便だと思います。
鈴木：確かに。でも環境への影響も心配ですね。

E45: 一番大切なものは何ですか。
【価値観について話す】
====================================

フレーズリスト
--------------
＊意見を求める
・あなたにとって、何が一番大切ですか。
・仕事を選ぶ時に一番大切にすることは何ですか。給料ですか、仕事内容ですか。
・宝くじが当たったら、何に使いますか。

＊意見を述べる
・私にとって一番大切なものは、家族です。
・私にとって最も大切なものは友達です。
・友達さえいれば、どんなに大変なことがあっても頑張れると思います。
・高い給料でも、健康でなければ意味がないと思います。
・高い給料をもらうより、体の方が大切だと思います。

＊理由を述べる
・なぜなら･･･／どうしてかというと･･･／なぜかというと･･･
・体さえ健康なら、どんなことにも挑戦できるからです。

単語リスト
----------
価値観、考え方、出世、仕事内容、健康、財産、（お金の）使い方


====== 丁寧版 ======

【価値観について話す】
田中：鈴木さんにとって、一番大切なものは何ですか。
鈴木：私にとっては家族が一番大切です。
田中：素晴らしいですね。なぜ家族が一番大切なんですか。
鈴木：家族さえいれば、どんな困難も乗り越えられると思うからです。

【仕事の価値観について】
鈴木：田中さんは、仕事を選ぶ時に何を重視されますか。
田中：私は仕事内容を重視します。給料よりも大切だと思います。
鈴木：それはなぜですか。
田中：やりがいがなければ、高い給料でも意味がないと思うからです。

【健康の価値について】
田中：私は健康が何より大切だと思います。
鈴木：確かに。体さえ健康なら、どんなことにも挑戦できますね。
田中：そうなんです。どんなに高い給料でも、健康でなければ意味がありません。
鈴木：おっしゃる通りですね。

E46: 荷物を間違えちゃったんだ。
【体験談について話す】
====================================

フレーズリスト
--------------
＊トラブルの体験を話す
・旅行から帰ってきてスーツケースをあけたら、見覚えのないものばかりが入っていたんだ。
・その時初めて気付いて！違う人のスーツケースだったんだよ。
・同じスーツケースだったからあけるまで気付かなくって。
・自分の荷物が届くまで１週間もかかったんだよ。

＊トラブル続きの経験
・この前海外旅行に行った時のことなんだけど･･･
・買ったばかりの傘をなくすし、電車は遅れるし、トラブル続きだったんだ。
・おまけに、お土産を買ったのに、ホテルにおいてきてしまったし。

＊面接の日のトラブル
・この前面接の日に寝坊して、慌てて着替えて駅まで走って行って･･･
・電車の時間に間に合わなくて、タクシーで行ったんだ。
・せっかく新しいスーツで行ったのに、汗をすごくかいちゃって。
・おまけに、シャツにコーヒーの染みが付いちゃたり、本当にトラブル続きだったよ。

単語リスト
----------
取り違える、寝坊する、終わりよければ全て良し


====== 丁寧版 ======

【旅行のトラブルについて話す】
田中：先日の旅行で大変なことがあったんです。
鈴木：どうされたんですか。
田中：スーツケースを間違えてしまいまして。帰ってきて開けたら、見覚えのないものばかりで。
鈴木：それは大変でしたね。どうなさったんですか。

【詳細を説明する】
田中：その時初めて気付いて、空港に連絡しました。
鈴木：自分の荷物は見つかったんですか。
田中：はい、でも届くまで1週間もかかったんです。
鈴木：それはお困りでしたね。

【面接のトラブルについて話す】
鈴木：実は先日、面接の日に寝坊してしまいまして。
田中：それは大変でしたね。
鈴木：慌てて着替えて駅まで走ったんですが、電車に間に合わなくて。
田中：結局どうなさったんですか。
鈴木：タクシーで行きました。本当にトラブル続きで大変でした。

補足：各エピソードの活用方法
====================================

【丁寧版とカジュアル版の使い分け】
・丁寧版：ビジネスシーン、初対面の人、目上の人との会話
・カジュアル版：友達同士、同僚との気軽な会話、家族との会話

【学習のポイント】
1. まず基本フレーズを覚える
2. 丁寧版で正式な表現を学ぶ
3. カジュアル版で日常会話を練習する
4. 状況に応じて使い分けられるようにする

【練習方法】
1. ロールプレイング：二人一組で役を交代しながら練習
2. 状況設定：異なる場面を想定して表現を変えてみる
3. 単語の入れ替え：基本文型を使って単語を変えて練習
4. 自由会話：学んだ表現を使って自由に会話を展開する

【E41-E50の特徴】
このセクションでは、より高度な社会問題や価値観について話す表現が含まれています。
・E41-E42: 文化と反省（習慣、後悔）
・E43-E45: 社会と価値観（現代社会の問題、必需品、価値観）
・E46-E48: 経験と説明（体験談、犯罪、料理）
・E49-E50: 社会問題（異常気象、賛否両論）
