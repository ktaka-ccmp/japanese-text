E31: たしか、ブラジルのお祭りですよね。
【世界のお祭りについて話す】
====================================

フレーズリスト
--------------
＊お祭りについてたずねる
・リオのカーニバルというお祭りを知っていますか。
・知りません、どんなお祭りですか。
・名前だけ知っています。有名ですよね。
・詳しいことはわからないんですけど、たしかブラジルのお祭りですよね。

＊お祭りについて説明する
・ブラジルのリオ・デ・ジャネイロで行われるサンバパレードが有名な祭りです。
・毎年2月ごろ行われ、世界中から多くの観光客が集まります。
・メキシコの死者の日っていう祭りは日本のお盆に似ていますよね。
・日本ではお盆の時期に全国でいろいろなお祭りが行われます。盆踊りが有名です。

単語リスト
----------
祭りの時期、成長を願う、収穫を祝う、豊作に感謝する
先祖の供養をする、盆踊りを踊る、特別な衣装を着る
行列、パレードを行う、屋台


====== 丁寧版 ======

【お祭りについて話す】
田中：リオのカーニバルというお祭りをご存じですか。
鈴木：名前だけは知っています。ブラジルのお祭りですよね。
田中：はい、そうです。リオ・デ・ジャネイロで毎年行われるサンバパレードが有名です。
鈴木：世界的に有名ですよね。一度は見てみたいです。

【他の国のお祭りについて】
田中：メキシコの死者の日という祭りをご存じですか。
鈴木：いいえ、知りません。どんなお祭りですか。
田中：日本のお盆に似ていて、先祖の供養をする祭りなんです。
鈴木：そうなんですね。興味深いですね。

【日本のお祭りについて説明する】
鈴木：日本ではお盆の時期にどんなお祭りがありますか。
田中：全国でいろいろな祭りが行われます。盆踊りが特に有名です。
鈴木：盆踊りですか。参加したことはありますか。
田中：はい、子供の頃によく参加していました。


====== カジュアル版：友達との会話 ======

鈴木：世界的に有名だよね。一度見てみたいな。

E32: それは縁起がいいね。
【迷信について話す】
====================================

フレーズリスト
--------------
＊迷信や言い伝えを紹介する
・昔から、夜新しい靴を出してはいけないと言われています。
・子供のころ、よく母に「食べてすぐ横になってはいけない、牛になるよ。」と言われていました。
・方角にまつわる話なんですけど、北枕で寝ると良くないことが起こると言われています。
・くしゃみをすると、誰かがうわさしているとよく言われます。
・受験生の周りで「落ちる」や「すべる」など不合格を連想させる言葉を言うのは縁起が悪いとされています。
・私は大切な試験や面接の前の日は、げんかつぎで豚カツやカツ丼を食べます。

＊自分の考えを述べる
・私はそんなの信じてないんですけどね。
・別に信じているわけではないけど、何となく北枕はさけるようにしています。
・信じているわけではないけど、四つ葉のクローバーを見つけるとうれしくなります。
・気にはしていないですけど、歩いていて黒猫が前を横切ると嫌な気持ちになります。

単語リスト
----------
動物にまつわる話、縁起が良い、めでたい
縁起が悪い、不吉な、語呂合わせ、げんをかつぐ


====== 丁寧版 ======

【迷信について話す】
田中：日本には「北枕で寝ると良くない」という言い伝えがあります。
鈴木：そうなんですか。信じていらっしゃいますか。
田中：別に信じているわけではないですが、何となく避けるようにしています。
鈴木：なるほど。私も似たようなことがありますよ。

【げんかつぎについて】
鈴木：田中さんは大事な試験の前に何かされますか。
田中：はい、げんかつぎでカツ丼を食べます。「勝つ」という語呂合わせです。
鈴木：面白いですね。私も受験の時はそういうことをしていました。
田中：効果があるかはわかりませんが、気持ちが落ち着きますね。

【言い伝えについて意見を述べる】
田中：くしゃみをすると誰かがうわさしているって聞いたことありますか。
鈴木：はい、よく聞きます。でも信じてはいないですね。
田中：私もです。でも何となく気になってしまいます。
鈴木：わかります。そういうものですよね。


====== カジュアル版：友達との会話 ======

【迷信について話す】
田中：日本って「北枕で寝ると良くない」っていう言い伝えあるよね。

鈴木：なるほどね。俺も似たようなことあるよ。

【げんかつぎについて】

鈴木：面白いね。俺も受験の時やってたわ。
田中：効果あるかわかんないけど、気持ちが落ち着くんだよね。

【言い伝えについて意見を述べる】

E33: 大丈夫ですか。
【身の周りでよくある出来事（災難）について話す】
====================================

フレーズリスト
--------------
＊アクシデントについて話す
・朝、寝ぼけていて、階段から落ちて、足をねんざしました。
・カップラーメンにお湯を注いでいたら、手元がくるって、こぼしてしまったんです。
・料理していたら、ついフライパンに当たってしまったんです。
・ぼうっと歩いていたら、自転車とぶつかりそうになって。よけようとして転んだんです。
・洗濯物を取り入れようとして、ハチにさされたんです。まさかこんなことになるなんて・・・

単語リスト
----------
＊アクシデント
こける、転ぶ、ひっくり返る、階段から落ちる
車にひかれる、犬にかまれる、虫に刺される
コーヒーをこぼす、料理を焦がす、雪道で滑る

＊怪我
ねんざする、くじく、ひねる、打つ、痛める
切る、すりむく、骨が折れる、足にあざができる
すり傷ができる、火傷する


====== 丁寧版 ======

【怪我について心配する】
田中：鈴木さん、足を引きずっていらっしゃいますが、大丈夫ですか。
鈴木：ありがとうございます。実は階段から落ちて、足をねんざしてしまいまして。
田中：それは大変でしたね。病院には行かれましたか。
鈴木：はい、昨日行ってきました。しばらく安静にしていれば治るそうです。

【アクシデントについて話す】
鈴木：今朝、料理中にやけどをしてしまったんです。
田中：それは大変ですね。ひどいですか。
鈴木：軽いやけどなので、大丈夫です。フライパンに手が当たってしまって。
田中：気をつけてくださいね。お大事になさってください。

【災難について同情する】
田中：洗濯物を取り入れる時にハチに刺されてしまったんです。
鈴木：それは災難でしたね。痛かったでしょう。
田中：はい、とても痛かったです。まさかこんなことになるとは思いませんでした。
鈴木：本当に気の毒です。早く良くなるといいですね。


====== カジュアル版：友達との会話 ======

【怪我について心配する】
田中：鈴木君、足引きずってるけど、大丈夫？
鈴木：ありがとう。実は階段から落ちて、足ねんざしちゃって。
田中：それはやばいね。病院行った？
鈴木：うん、昨日行ってきた。しばらく安静にしてれば治るって。

【アクシデントについて話す】
鈴木：今朝、料理中にやけどしちゃったんだ。
田中：マジで？ひどい？
鈴木：軽いやけどだから大丈夫。フライパンに手が当たっちゃって。
田中：気をつけてね。お大事に。

【災難について同情する】

鈴木：それは災難だったね。痛かったでしょ。

鈴木：本当に気の毒。早く良くなるといいね。

E34: 最近、ゴルフにこっています。
【今一番興味のあること、最近興味があることについて話す】
====================================

フレーズリスト
--------------
＊趣味について話す
・ノバーズの鈴木の大ファンなんです。
・いつからなんですか。
・中学生の頃からずっとです。
・一度コンサートに行って以来、ずっとです。

＊きっかけをたずねる・答える
・好きになったきっかけは何だったんですか。
・もともと兄がファンで、家にたくさんCDがあったんです。
・それで何となく私も聞くようになって。
・映画でアクションシーンを見てからすっかりファンになって。

＊最近の趣味について話す
・私は、最近、写真に凝っています。
・いつごろから撮り始めたんですか。
・最近なんです。半年ぐらい前からです。
・何か始めたきっかけがあったんですか。
・海外旅行に行って、大自然の美しさに感動してからです。
・あのすばらしさは言葉では表現できません。
・今、釣りに凝っています。
・初めて行った時に大きいのが釣れて嬉しくて・・・それ以来よく行きます。
・最近、ヨガが気になります。
・健康の為に始めてみようかと思っています。

大ファン、熱狂的なファン、切手収集家、コレクター

E35: センスがいいですよね。
【ファッションや衣服について話す】
====================================

フレーズリスト
--------------
＊ファッションについて話す
・どの年代にもコンサバタイプの人はいますが、基本的に落ち着いた雰囲気を好む人が多いです。
・洋服をデパートなどで購入することが多いです。
・私は若い頃は、最新のファッションが好きでした。
・今は流行物も時々買いますが、流行が過ぎると着られなくなってしまうので、質が良くてシンプルな長く着られる物を選ぶことが多いです。

単語リスト
----------
カジュアル、スポーティ、クラシック、レトロ、コンサバ、古着
ドレッシーな、きれいめな、品がいい、きちんとした
流行の、最新の、センスがいい、おしゃれな、今年らしい
質がいい、安っぽい、ビジネスシーンにふさわしい服装
動きやすい格好、着心地がいい、実用性がある
着まわせる、合わせやすい、着こなし、合わせ方


====== 丁寧版 ======

【ファッションの好みについて話す】
田中：鈴木さんはどんなファッションがお好きですか。
鈴木：私は基本的にシンプルで落ち着いた雰囲気のものが好きです。
田中：わかります。私も最近は質の良い長く着られるものを選ぶようにしています。
鈴木：流行物も素敵ですが、長く着られるものの方が経済的ですよね。

【服の選び方について話す】
鈴木：田中さんはどこで洋服を購入されることが多いですか。
田中：デパートで購入することが多いです。品質がいいので。
鈴木：そうですね。私も同じです。試着できるのもいいですよね。
田中：はい、サイズ感も確認できますし、安心です。

【ファッションセンスについて褒める】
田中：鈴木さん、今日のコーディネート、センスがいいですね。
鈴木：ありがとうございます。シンプルですが、着まわしやすいんです。
田中：とても品があって素敵です。参考にさせてください。
鈴木：恐縮です。ありがとうございます。

鈴木：流行物も素敵だけど、長く着られる方が経済的だよね。

【服の選び方について話す】

鈴木：そう言ってもらえると嬉しいよ。ありがとう。

E36: 期待はずれだった。
【映画について批評する】
====================================

フレーズリスト
--------------
＊感想を求める
・よかったよね。どう思った。
・あの主役の人、演技うまいと思わない。
・最初から、結末が／ラストが想像できたよね。
・よくある話だったけど、アクションがすごかったね。
・展開がすごく早くておもしろかったね。

＊意見を言う
・展開が早くて、引き込まれた。
・難しかったけど、なんか考えさせられたな。
・話が単純すぎてつまらなかったよ。
・正直言って、あんまりおもしろくなかった。
・思っていたほどおもしろくなかった。
・話題になっていたわりには、ありがちな展開だった。
・期待はずれだった。あれはひどかった。

単語リスト
----------
重い、不思議な、ありきたりな、ありがちな
単純な、複雑な、切ない、感動的な

【映画の感想を述べる】

田中：そうでしたか。私も見たのですが、少し複雑で難しかったです。
鈴木：確かに難しい部分もありましたが、考えさせられる内容でしたね。

【批評的な意見を述べる】
鈴木：話題の映画、どうでしたか。
田中：正直に申し上げると、期待していたほどではありませんでした。
鈴木：そうなんですか。私も似たような感想を持ちました。
田中：話題になっていた割には、ありがちな展開でしたね。

【肯定的な感想を述べる】
田中：アクションシーンが素晴らしかったですね。
鈴木：本当にそう思います。迫力がありました。
田中：主役の演技も良かったと思いませんか。
鈴木：はい、とても上手でした。感動的でしたね。

====================================
E38: ちょっとお伺いしたいんですが･･･
【電話で問い合せをする】
====================================

フレーズリスト
--------------
＊話を切り出す（電話のかけ方）
・すみません、ちょっとおたずねしたいんですが･･･
・時間の確認をしたいんですが…
・そちらまでの行き方を教えて頂きたいんですが…

＊電話での応対
・どちらのツアーにお申込み頂いておりますでしょうか。
・どういったご用件でしょうか。

＊詳細をたずねる
・このコースの料金を教えて頂きたいんですが･･･
・平日のコースと土日のコースの内容は同じでしょうか。
・中学生も参加できますか。
・キャンセル料はいつからかかりますか。
・この料金に、三日目の昼食も含まれていますでしょうか。
・これ以外にかかる費用はありますか。
・9時までに現地に着くバスはありますか。
・オプショナルコースは、後からつけることができますか。
・雨の場合はどうなりますか。

単語リスト
----------
受講期間、申込み期間、車中泊


====== 丁寧版 ======

【問い合わせの電話をかける】
田中：お忙しいところ恐れ入ります。ツアーについてお伺いしたいのですが。
スタッフ：はい、ありがとうございます。どちらのツアーでしょうか。
田中：週末の富士山ツアーについてです。
スタッフ：かしこまりました。どのようなご質問でしょうか。

【詳細を確認する】
田中：料金について教えていただけますでしょうか。
スタッフ：はい、こちらのコースは一名様25,000円となっております。
田中：その料金に昼食代も含まれていますでしょうか。
スタッフ：はい、全ての食事代が含まれております。

【追加の質問をする】
田中：キャンセル料はいつから発生しますでしょうか。
スタッフ：出発日の7日前からキャンセル料が発生いたします。
田中：承知いたしました。もう一つお伺いしたいのですが、雨天の場合はどうなりますでしょうか。
スタッフ：雨天でも決行いたしますが、悪天候の場合は中止になることもございます。


====== カジュアル版：友達との会話 ======

E39: UFOを見たことがあります。
【不思議な体験について話す】
====================================

フレーズリスト
--------------
＊不思議な体験について話す
・UFOを見たことがあります。
・本当ですか。どんな形でしたか。
・夜空に光るものが見えて、最初は飛行機かと思いました。
・でも、動きが不規則で、すごく速かったんです。
・信じられないかもしれませんが、本当なんです。

＊疑問を表す
・本当にそんなことがあったんですか。
・もしかして見間違いじゃないですか。
・ちょっと信じられないんですけど。

＊体験を強調する
・この目で見たんです。間違いありません。
・私も最初は信じられませんでした。
・周りの人も見ていたので、間違いないです。

単語リスト
----------
UFO、未確認飛行物体、宇宙人、不思議な現象
目撃する、体験する、信じられない、疑う


====== 丁寧版 ======

【不思議な体験について話す】
田中：実は以前、不思議なものを見たことがあるんです。
鈴木：本当ですか。どのようなものでしたか。
田中：夜空に光るものが見えて、最初は飛行機だと思ったんですが。
鈴木：それで、どうだったんですか。

【詳しく説明する】
田中：動きが不規則で、とても速かったんです。
鈴木：それは不思議ですね。UFOだったんでしょうか。
田中：わかりませんが、今まで見たことのないものでした。
鈴木：興味深いお話ですね。他に見た方はいらっしゃいましたか。

【体験を強調する】
田中：はい、周りにいた数人も見ていました。
鈴木：それでしたら、何か不思議な現象だったかもしれませんね。
田中：信じられないかもしれませんが、この目で見たんです。
鈴木：いえ、信じますよ。世の中には不思議なことがありますからね。

====================================
補足：各エピソードの活用方法
====================================

【丁寧版とカジュアル版の使い分け】
・丁寧版：ビジネスシーン、初対面の人、目上の人との会話
・カジュアル版：友達同士、同僚との気軽な会話、家族との会話

【学習のポイント】
1. まず基本フレーズを覚える
2. 丁寧版で正式な表現を学ぶ
3. カジュアル版で日常会話を練習する
4. 状況に応じて使い分けられるようにする

【練習方法】
1. ロールプレイング：二人一組で役を交代しながら練習
2. 状況設定：異なる場面を想定して表現を変えてみる
3. 単語の入れ替え：基本文型を使って単語を変えて練習
4. 自由会話：学んだ表現を使って自由に会話を展開する
