C01: 初めてお会いしますね。
【初対面の人に挨拶をする】
====================================

フレーズリスト
--------------
・初めてお会いしますね。
・はい、先週引っ越してきました。
・はい、シドニーから転勤してきました。
・どちらからいらっしゃったんですか。
・福岡です。
・以前は福岡に住んでいました。
・この辺はどうですか。
・とてもいいところですね。
・いつこちらにいらっしゃったんですか。
・1週間前です。
・東京本社に転勤になったんです。
・わからないことがあったら、気軽に声をかけてください。
・ありがとうございます。
・よろしくお願いします。

単語リスト
----------
引っ越し、転勤、留学、営業、東京本社、福岡支店
先週、今週、来週


====== 丁寧版 ======

【初めての挨拶】
田中：初めてお会いしますね。いつこちらにいらっしゃったんですか。
王さん：先週引っ越してきました。よろしくお願いします。
田中：こちらこそ。どちらからいらっしゃったんですか。
王さん：上海から転勤してきました。

【場所について聞く】
田中：この辺はどうですか。
王さん：とてもいいところですね。便利で静かです。
田中：それは良かったです。わからないことがあったら、気軽に声をかけてください。
王さん：ありがとうございます。助かります。

【転勤について】
王さん：田中さんは以前からこちらにいらっしゃったんですか。
田中：はい、東京本社には3年前から勤務しています。
王さん：そうなんですね。いろいろ教えていただけると嬉しいです。
田中：もちろんです。何でも聞いてくださいね。


====== カジュアル版：職場の同僚との会話 ======

【初めての挨拶】
田中：初めまして。いつ来たの？
王さん：先週引っ越してきたばかりなんだ。よろしく。
田中：よろしく。どこから？
王さん：上海から転勤してきたんだ。

【場所について聞く】
田中：この辺どう？
王さん：すごくいいね。便利で静か。
田中：良かった。わかんないことあったら、気軽に聞いてね。
王さん：ありがとう。助かる。

【転勤について】
王さん：田中さんは前からここにいたの？
田中：うん、東京本社には3年前から。
王さん：そうなんだ。いろいろ教えてね。
田中：もちろん。何でも聞いて。

C02: プリンスホテルへはどう行けばいいですか。
【交通機関を使った目的地への行き方をたずねる】
====================================

フレーズリスト
--------------
・ショッピングセンターへはどう行けばいいですか。
・丸ノ内線の新宿行きに乗ってください。
・銀座で銀座線に乗り換えてください。
・水族館へ行きたいんですが。
・御堂筋線の天王寺行きに乗ってください。
・本町で中央線に乗り換えてください。
・本町から五つ目の駅です。
・五つ目の駅で降りてください。
・桜橋で54番のバスに乗ってください。
・山手線の品川方面行きに乗ってください。

単語リスト
----------
～線、～行き・～方面、乗る、乗り換える、降りる
乗客、駅員、一つ目、二つ目、三つ目、四つ目


====== 丁寧版 ======

【目的地への行き方を聞く】
王さん：すみません、プリンスホテルへはどう行けばいいですか。
駅員：丸ノ内線の新宿行きに乗ってください。
王さん：乗り換えは必要ですか。
駅員：はい、銀座で銀座線に乗り換えてください。

【駅の数を確認する】
王さん：何個目の駅で降りればいいですか。
駅員：銀座から三つ目の駅になります。
王さん：わかりました。ありがとうございます。
駅員：気をつけて行ってらっしゃいませ。

【バスの行き方】
王さん：水族館へ行きたいんですが、どう行けばいいですか。
駅員：桜橋駅で54番のバスに乗ってください。
王さん：バス停はどこにありますか。
駅員：駅を出て右側にあります。


====== カジュアル版：友達との会話 ======

【目的地への行き方を聞く】
王さん：ねえ、プリンスホテルってどう行くの？
友達：丸ノ内線の新宿行きに乗って。
王さん：乗り換えいる？
友達：うん、銀座で銀座線に乗り換えて。

【駅の数を確認する】
王さん：何個目の駅で降りればいい？
友達：銀座から三つ目だよ。
王さん：わかった。ありがとう。
友達：気をつけてね。

【バスの行き方】
王さん：水族館行きたいんだけど、どう行く？
友達：桜橋駅で54番のバス乗って。
王さん：バス停どこ？
友達：駅出て右側。

C03: 面接に行く。
【できることについて話す】
====================================

フレーズリスト
--------------
・料理ができますか。
・はい、できます。
・はい、少しだけ。
・はい、上手くないですけど...
・何が作れますか。
・何語が話せますか。
・英語とスペイン語が少し（話せます）。
・英語を話すことができます。
・どのくらい話せますか。
・日常会話ができます。
・挨拶程度です。
・漢字を読めます。
・何か楽器ができますか。
・はい、ドラムとギターができます。
・したことがないです。
・何のスポーツが教えられますか。
・野球とソフトボールが教えられます。
・上手に絵が描けますか。
・まあまあです。
・上手ではないですが・・・

単語リスト
----------
絵/描ける、ギター/弾ける、パソコン/使える
運転/できる、料理/できる


====== 丁寧版：面接 ======

【できることを聞く】
面接官：料理はできますか。
王さん：はい、できます。中華料理が得意です。
面接官：何が作れますか。
王さん：餃子、炒飯、麻婆豆腐などが作れます。

【語学力について】
面接官：何語が話せますか。
王さん：中国語、日本語、英語が話せます。
面接官：日本語はどのくらい話せますか。
王さん：日常会話とビジネス会話ができます。

【その他のスキル】
面接官：パソコンは使えますか。
王さん：はい、Word、Excel、PowerPointが使えます。
面接官：運転はできますか。
王さん：はい、できます。免許を持っています。


====== カジュアル版：友達との会話 ======

【できることを聞く】
友達：料理できる？
王さん：うん、できるよ。中華得意。
友達：何作れるの？
王さん：餃子、炒飯、麻婆豆腐とかかな。

【語学力について】
友達：何語話せるの？
王さん：中国語、日本語、英語。
友達：日本語どのくらい話せる？
王さん：日常会話とビジネス会話は大丈夫。

【その他のスキル】
友達：パソコン使える？
王さん：うん、Word、Excel、PowerPoint使えるよ。
友達：運転できる？
王さん：うん、免許持ってる。

C04: ピザを注文したいんですが。
【電話で食べ物を注文する】
====================================

フレーズリスト
--------------
・ピザを注文したいんですが。
・配達をお願いします。
・30分ほどかかりますが、よろしいですか。
・電話番号からお願いします。
・ご注文をお願いします。
・マルガリータのLサイズをお願いします。
・生地をお選びください。
・クリスピーで。
・他にご注文はございますか。
・コーンのトッピングをお願いします。
・いいえ、以上です。

単語リスト
----------
Sサイズ、Mサイズ、Lサイズ、生地、クリスピー
レギュラー、トッピング、サイドメニュー


====== 丁寧版 ======

【注文を始める】
王さん：もしもし、ピザを注文したいんですが。
店員：ありがとうございます。配達ですか、それともお持ち帰りですか。
王さん：配達をお願いします。
店員：かしこまりました。お電話番号からお願いします。

【ピザを注文する】
店員：ご注文をどうぞ。
王さん：マルガリータのLサイズをお願いします。
店員：生地はクリスピーとレギュラーがございますが。
王さん：クリスピーでお願いします。

【追加注文】
店員：他にご注文はございますか。
王さん：コーンのトッピングを追加してください。
店員：かしこまりました。30分ほどでお届けいたします。
王さん：よろしくお願いします。


====== カジュアル版：友達同士での注文 ======

【注文を始める】
王さん：もしもし、ピザ注文したいんだけど。
店員：ありがとうございます。配達ですか、お持ち帰りですか。
王さん：配達で。
店員：電話番号教えてください。

【ピザを注文する】
店員：何にしますか。
王さん：マルガリータのLサイズ。
店員：生地はクリスピーとレギュラーどっちがいいですか。
王さん：クリスピーで。

【追加注文】
店員：他に何か注文ありますか。
王さん：コーンのトッピング追加して。
店員：わかりました。30分くらいで届けます。
王さん：お願いします。

【家具について話す】
====================================

フレーズリスト
--------------
・新しい家はどうですか。
・とてもいいですよ。
・リビングに革のソファを買いました。
・他に何か買う予定ですか。
・ベッドを買おうと思っています。
・大きいテレビを買ったんだ。
・高かった？
・うん、ちょっと。
・ううん、そんなに。
・ううん、高くなかったよ。
・ううん、安かったよ。
・可愛いカーテンだね。
・先週買ったんです。
・イタリア製なんです。
・他に何か買う予定？
・新しいテーブルを買おうと思っています。
・リビングに観葉植物が欲しいんです。

単語リスト
----------
寝室、キッチン/台所、リビング/居間、ラグ/じゅうたん


====== 丁寧版 ======

田中：新しい家はどうですか。
王さん：とてもいいですよ。広くて明るいです。
田中：それは良かったですね。家具は揃いましたか。

【購入予定について】
田中：他に何か買う予定ですか。
王さん：ベッドと新しいテーブルを買おうと思っています。
田中：どんなテーブルをお探しですか。
王さん：木製の大きめのダイニングテーブルが欲しいです。

【家具について感想を述べる】
王さん：田中さんのカーテン、とても可愛いですね。
田中：ありがとうございます。先週買ったんです。
王さん：どちらで買われたんですか。
田中：IKEAで買いました。イタリア製なんですよ。


====== カジュアル版：友達との会話 ======

王さん：まだだけど、リビングに革のソファ買った。

友達：どんなテーブル探してるの？

C06: 最近どうですか。
【知人と短い会話をする】
====================================

フレーズリスト
--------------
・やあ、翔。
・やあ。
・最近どう？
・元気だよ。
・そっちは？
・仕事はどう？
・忙しいよ。
・昨日の試合を見た？
・うん、見たよ。いい試合だったね。
・ううん、昨日は外出したから見られなかったよ。
・いい天気だね。
・そうだね。暖かいね。
・家族は元気？
・うん、元気だよ。
・ごめん、そろそろ行かないと。
・じゃあ、またゆっくり。
・ごめん。そろそろ行くよ。またね。

単語リスト
----------
忙しい、相変わらず、元気、天気


====== 丁寧版 ======

【偶然会った時】
王さん：田中さん、こんにちは。
田中：やあ、王さん。最近どうですか。
王さん：元気です。田中さんは？
田中：忙しいですが、元気ですよ。

【近況を話す】
王さん：お仕事はどうですか。
田中：相変わらず忙しいです。王さんは？
王さん：私も忙しいですが、充実しています。
田中：それは良いですね。

【別れる】
田中：すみません、そろそろ会社に戻らないと。
王さん：そうですか。では、また今度ゆっくりお話ししましょう。
田中：はい、ぜひ。それでは失礼します。
王さん：お疲れ様です。


====== カジュアル版：友達との会話 ======

【偶然会った時】
王さん：やあ、翔。
友達：やあ。最近どう？
王さん：元気。そっちは？
友達：忙しいけど元気だよ。

【近況を話す】
王さん：仕事どう？
友達：相変わらず忙しい。王さんは？
王さん：俺も忙しいけど、充実してる。
友達：それはいいね。

【別れる】
友達：ごめん、そろそろ行かないと。
王さん：そっか。じゃあまたゆっくり話そう。
友達：うん、また連絡するよ。じゃあね。
王さん：またね。

C08: 待ち合わせはどこにする？
【友達と会う約束をする】
====================================

フレーズリスト
--------------
・明日の夜、サッカーを見に行かない？
・いいね。行こう。
・ごめん。明日はちょっと…
・24日の水曜日はどう？
・行けるよ。
・何時にする？
・待ち合わせはどこにする？
・今日ご飯食べに行かない？
・いいね。何食べる？
・何時にする？
・7時半はどう？
・ごめん。7時半まで、仕事なんだ。
・じゃあ、8時半でどう？
・じゃあそれで。
・待ち合わせはどこにする？
・駅の改札口で会おうよ。
・映画館の前はどう？
・じゃあ7時半に映画館の前ね。

単語リスト
----------
食事に行く、飲みに行く、サッカーを見に行く


====== 丁寧版（職場の先輩と） ======

【誘う】
王さん：田中さん、今週の土曜日、食事に行きませんか。
田中：いいですね。どこか行きたいところはありますか。
王さん：新しくオープンした中華料理店はどうですか。
田中：いいですね。何時にしましょうか。

【時間を決める】
王さん：6時半はいかがですか。
田中：すみません、6時半はちょっと難しいです。
王さん：それでは、7時はどうですか。
田中：7時なら大丈夫です。

【場所を決める】
王さん：待ち合わせはどこにしましょうか。
田中：駅の改札口で会いましょうか。
王さん：はい、それでお願いします。
田中：では、土曜日の7時に駅の改札口で。


====== カジュアル版：友達との会話 ======

【誘う】
王さん：明日の夜、飯食いに行かない？
友達：いいね。何食べる？

友達：いいね。何時にする？

【時間を決める】
王さん：6時半は？
友達：ごめん、6時半まで仕事なんだ。
王さん：じゃあ、7時は？
友達：7時なら大丈夫。

【場所を決める】
王さん：待ち合わせどこにする？

C09: 両替したいのですが。

====================================

フレーズリスト
--------------
・いらっしゃいませ。
・円をUSドルに両替したいのですが。
・手数料はいくらかかりますか。
・5ドルです。
・2パーセントです。
・いくら両替なさいますか。
・いくら両替しますか。
・50,000円をUSドルに両替してください。
・25,000円をトラベラーズチェックに換えてください。
・パスポートを見せていただけますか。
・はい。
・かしこまりました。700ドルです。
・こちらにサインをお願いします。

単語リスト
----------
1,000 = 千、10,000 = 一万、20,000 = 二万、30,000 = 三万
40,000 = 四万、50,000 = 五万、60,000 = 六万、70,000 = 七万
80,000 = 八万、90,000 = 九万、100,000 = 十万


====== 丁寧版 ======

【両替を申し込む】
王さん：すみません、円をUSドルに両替したいのですが。
銀行員：かしこまりました。いくら両替なさいますか。
王さん：50,000円お願いします。
銀行員：手数料は5ドルになります。

【本人確認】
銀行員：パスポートを見せていただけますか。
王さん：はい、どうぞ。
銀行員：ありがとうございます。少々お待ちください。
王さん：はい。

【両替完了】
銀行員：お待たせいたしました。50,000円で700ドルになります。
王さん：はい。
銀行員：こちらにサインをお願いします。
王さん：わかりました。

====================================
補足：各エピソードの活用方法
====================================

【丁寧版とカジュアル版の使い分け】
・丁寧版：ビジネスシーン、初対面の人、目上の人との会話、フォーマルな場面
・カジュアル版：友達同士、同僚との気軽な会話、家族との会話、インフォーマルな場面

【中国人学習者向けの特徴】
・漢字を積極的に使用（中国人学習者は漢字の認識が速い）
・実用的な場面を重視（転勤、引っ越し、面接、買い物など）
・丁寧語とカジュアル語の違いを明確に提示

【学習のポイント】
1. まず基本フレーズを覚える
2. 丁寧版で正式な表現を学ぶ
3. カジュアル版で日常会話を練習する
4. 状況に応じて使い分けられるようにする
5. 同じ意味でも異なる表現を比較して理解を深める

【練習方法】
1. ロールプレイング：二人一組で役を交代しながら練習
2. 状況設定：異なる場面を想定して表現を変えてみる
3. 単語の入れ替え：基本文型を使って単語を変えて練習
4. 自由会話：学んだ表現を使って自由に会話を展開する
5. 録音練習：自分の発音を録音して確認する

【レベル3の特徴】
このレベルでは、日常生活で必要な基本的なコミュニケーションができるようになります。
・C01-C03: 挨拶と自己紹介、道案内、能力の説明
・C04-C06: 注文、買い物、短い会話
・C07-C09: 商品購入、約束、両替などの実用的な場面
